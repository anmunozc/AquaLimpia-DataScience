# AquaLimpia-DataScience

Proyecto de Ciencia de Datos desarrollado para el diagnóstico operacional y ambiental de las plantas de tratamiento de AquaLimpia S.A.

## Objetivos
* Analizar patrones operacionales y eficiencia de tratamiento.
* Visualizar indicadores ambientales críticos (DBO).
* Implementar un flujo de trabajo reproducible y modular.
* Garantizar trazabilidad mediante control de versiones.

## Herramientas utilizadas
* **Lenguaje:** Python
* **Librerías:** Pandas, NumPy, Matplotlib, Seaborn, Plotly, SciPy, Joblib
* **Entorno:** Google Colab
* **Gestión:** Git / GitHub

## Estructura del Repositorio
* `aqualimpia.ipynb`: Notebook principal con el flujo analítico completo.
* `/scripts/`: Módulos de Python reutilizables (`limpieza.py`) para validación y normalización.
* `/dashboard/`: Recursos gráficos (.png y .pdf) que detallan el flujo de trabajo.
* `dataset_set_A_aguas_residuales.xlsx`: Dataset original.
* `dataset_procesado.pkl`: Dataset serializado para eficiencia.

## Metodología
Este proyecto sigue buenas prácticas de ingeniería de software aplicadas a la Ciencia de Datos, promoviendo la modularidad, la reproducibilidad y la gestión de datos mediante serialización.
