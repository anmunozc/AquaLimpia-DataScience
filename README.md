# AquaLimpia-DataScience
Proyecto de Ciencia de Datos desarrollado en Google Colab para el análisis operacional y ambiental de AquaLimpia S.A.

**Objetivos**

* Analizar patrones operacionales.
* Visualizar indicadores ambientales.
* Detectar dashboard exploratorios.
* Aplicar principios de reproducibilidad y control de versiones.

**Herramientas utilizadas**

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Ploty
* Google Colab
* GitHub
* SciPy
* Joblib

**Contenido del repositorio**

* Notebook principal.
* Dataset utilizado
* Capturas de graficos
* Scripts auxiliares.
